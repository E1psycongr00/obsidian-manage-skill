# 개인 위키 메모 조각

- 읽고 적는 양이 늘수록 "언젠가 정리해야 할 것"이 쌓인다는 느낌이 더 강해진다.
- note 앱이 기억 보조보다 미완성 프로젝트 창고처럼 느껴질 때가 많다.
- 다시 안 보는 이유가 검색 문제라기보다 "왜 지금 다시 봐야 하는지" 맥락이 없어서인 것 같다.
- atomic note 조언을 따라 잘게 쪼개면 오히려 메모가 더 살아 있지 않고, 문맥을 잃는 느낌이 든다.
- 미래의 나를 위해 쓴다고 하지만 실제로는 현재의 불안을 잠깐 진정시키는 용도로 적는 경우가 많다.
- 책 하이라이트, 클리핑, 회의 메모, 아이디어 조각이 다 같은 곳에 쌓이는데 서로 연결되기보다 미해결 목록처럼 남는다.
- 저장은 많이 했는데 사고가 전진했다는 느낌은 적다.
